# Ivan Rozinko — Critical Systems Engineering Leader

Personal site and professional positioning for Ivan Rozinko.

## Positioning

Critical Systems / Platform / Reliability Engineering Leader working across software, firmware, cloud, QA strategy, fintech/Web3 risk, AI reliability, and regulated systems.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy on Vercel

1. Import this repository in Vercel.
2. Framework preset: Next.js.
3. Build command: `npm run build`.
4. Output: default Next.js output.
